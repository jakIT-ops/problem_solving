### Big-O Quick Reference

- Array traversal: O(n)

- Sorting: O(n log n)

- Hash-map lookup: O(1) average

- BFS/DFS: O(V + E)

- Dijkstra (with heap): O((V+E) log V)

- Segment tree ops: O(log n)

### DP Patterns list

- 0/1 knapsack, unbounded knapsack, LIS via patience sorting, string DP (edit distance), DP on trees, bitmask DP.

### Graph algorithms

- BFS: shortest unweighted

- DFS: component / topo / backtracking

- Dijkstra: non-negative weights

- Bellman-Ford: negative edges

- A\*: heuristic-guided search
